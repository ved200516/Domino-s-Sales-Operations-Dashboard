# Domino's-Sales-Operations-Dashboard :
Developed an interactive Power BI dashboard using a Domino’s pizza orders dataset to analyze sales performance, revenue trends, order behavior, cancellations, and delivery distance. The dashboard highlights key KPIs, top pizza variants, peak order hours, and store-wise revenue to support data-driven business decisions.

Objectives:

--> Analyze total orders and total revenue

--> Track cancellation rate and order status

--> Identify top-performing pizza variants

--> Find peak ordering hours

--> Compare store-wise revenue performance

--> Analyze delivery distance impact

Key KPIs:

-->Total Orders

-->Total Revenue

-->Average Order Value (AOV)

-->Cancellation Rate

-->Average Delivery Distance

Dashboard Insights:

-->Generated ₹114.96K revenue from 188 orders

-->Peak orders occur during lunch and evening hours

-->Cancellation rate highlights areas for service improvement

-->Store-wise analysis helps identify top and low-performing stores
